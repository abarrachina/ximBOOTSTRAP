Welcome to ximBootstrap

This is module for Ximdex to build high customizable project in a easy way. XimBootstrap allows you give the first step in Ximdex and start to know all about this CMS: ximlets, XSL templates, servers, and sections.

XimBootstrap is based on Bootstrap framework tu build the layout. To install ximdex with bootstrap moduloe follow these steps.

$ Download Ximdex from the [main repo](https://github.com/OXE/ximdex) (or from [this fork](https://github.com/abarrachina/ximdex))
$ Download ximBOOTSTRAP
$ Untar Ximdex into your apache directory. /var/www/
$ Untar ximBOOTSTRAP into _modules_ directory, inside ximdex, i.e /var/www/ximdex/modules/
$ Install ximdex: install/install.sh
$ Install ximBOOTSTAP module: install/module.sh install ximBOOTSTRAP
$ Log in your new Ximdex. 
$ Go to the second tab in the left panel, to change Administrator Rol: Add permissions to _create Bootstrap Project_
$ Go back to first tab in the left panel and select create bootstrap project
