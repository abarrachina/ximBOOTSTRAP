Welcome to ximBootstrap

This is module for Ximdex to build high customizable project in a easy way. XimBootstrap allows you give the first step in Ximdex and start to know all about this CMS: ximlets, XSL templates, servers, and sections.

XimBootstrap is based on Bootstrap framework tu build the layout. To install ximdex with bootstrap moduloe follow these steps.

  1. Download Ximdex from the main repo (or from this fork)
  2. Download ximBOOTSTRAP
  3. Untar Ximdex into your apache directory. /var/www/
  4. Untar ximBOOTSTRAP into modules directory, inside ximdex, i.e /var/www/ximdex/modules/
  5. Install ximdex: install/install.sh
  6. Install ximBOOTSTAP module: install/module.sh install ximBOOTSTRAP
  7. Log in your new Ximdex.
  8. Go to the second tab in the left panel, to change Administrator Rol: Add permissions to create Bootstrap Project
  9. Go back to first tab in the left panel and select create bootstrap project
